Column-count [Многоколочность]

Свойство column-count управляет количеством колонок, на которые будет разбит контент элемента. 

            
               column-count: n; /*--- n - числовле значение --- */
               column-width: px/%/em/rem;
               column-gap: px/em/rem;

Можно указать одно целое положительное число, означающее количество колонок. Вместо числа допустимо ключевое слово auto.Тогда количество колонок будет зависеть от ширины, указанной в свойстве column-width. 

            
               p { column-count: 3; }
            
               p {
                  column-count: auto;
                  column-width: 250px;
               }

Если одновременно задать column-width и column-count браузер будет отображать столько колонок, сколько влезет в ширину родительского блока. В примере ниже мы можем при помощи слайдера управлять размером блока. Граница поможет нам визуализировать его размеры. 

            
               .text-wrapper {
                  column-count: 4;
                  column-width: 250px;
               }
