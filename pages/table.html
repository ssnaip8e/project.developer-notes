Table [Таблица]

Таблицы HTML позволяют веб-разработчикам упорядочивать данные в строки и столбцы.

Ширина таблицы по умолчанию подстраивается под контент внутри, если не задавать дополнительные CSS-свойства. Это приводит к определённым сложностям на адаптивных сайтах. Если контент не помещается на маленький экран, то таблица не сжимается, у неё появляется горизонтальный скролл. У этой проблемы есть несколько потенциальных решений: скрывать не первостепенную информацию для пользователей мобильных устройств или перестраивать отображение таблицы, например, при помощи свойства display.

Задать границы элементам tr, thead, tfoot и tbody нельзя, поэтому задавайте их тегам table, th> или td. 

        
      /* --- Теги --- */
        table    – создание таблицы
        caption  – заголовок таблицы
        tr       – формирование строки таблицы
        th       – создание ячейки шапки
        td       – создание ячейки

      /* --- Логические секции --- */

        thead    – секция шапки таблицы
        tbody    – секция тело таблицы
        tfoot    – секция подвала таблицы
      
      /* --- Атрибуты --- */
      
        colspan  – слияние ячеек по горизонтали
        rowspan  – слияние ячеек по вертикали
        colgroup – назначает группу столбцов для форматирования
        col      – задает свойства столбца в colgroup

Подсветка строк таблицы через одну. Это помогает считывать длинные таблицы, глазу есть за что зацепиться. Например, сделаем каждую вторую строку с коричневым фоном. Для этого понадобится всего одно CSS-правило с псевдоклассом :nth-child(). 

        
          tbody tr:nth-child(odd) { background-color: #663613; }

Можно сделать так, чтобы строка с заголовками колонок прилипала при прокрутке длинной таблицы. Это удобно, если данных много и пользователь может просто забыть, в какой колонке какие данные. 

        
        th {
          position: -webkit-sticky;
          position: sticky;
          top: 0;
          z-index: 1;
        }

Не забудьте добавить position: relative для родителя. Заодно подстрахуемся и сделаем прилипающими только заголовки в шапке таблицы. 

        
          table { position: relative; }

Задайте фон заголовкам, чтобы текст ячеек не был виден при прокрутке. А чтобы избавиться от линий между ячейками, зададим для всей таблицы свойство border-collapse: collapse. 

        
          table {
            position: relative;
            border-collapse: collapse;
          }
          
          thead th {
            position: -webkit-sticky;
            position: sticky;
            top: 0;
            z-index: 1;
            background-color: #FF8630;
          }

Пример (код после примера)

Lorem, ipsum dolor.
— 1 2 3 4
Lorem ipsum dolor sit amet.
A A1+A2 A3 A4
B B1 B2 B3 B4
C C1+D1 C2 D2 C4+D4
D D2 D3 
          
        /* --- html --- */
  
        /* создание таблицы */
      <table> 
          /* задаем заголовок таблице */
          <caption>Lorem, ipsum dolor.</caption> 
          /*участок семантики - шапка таблицы */
          <thead> 
              /* создание строки */
              <tr> 
                  /* создание ячейки-заголовка */
                  <th>—</th> 
                  <th>1</th>
                  <th>2</th>
                  <th>3</th>
                  <th>4</th>
              </tr>
          </thead>
          /*участок семантики - подвал таблицы */
          <tfoot> 
              <tr>
                  <th colspan="5">Lorem ipsum dolor sit amet.</th>
              </tr>
          </tfoot>
          /*участок семантики - тело таблицы */
          <tbody> 
              <tr>
                  /* создание ячейки простой */
                  <th>A</th> 
                  /* создание слияние ячейки с боковой по уровню */
                  <td colspan="2">A1+A2</td>
                  <td>A3</td>
                  <td>A4</td>
              </tr>
              <tr>
                  <th>B</th>
                  <td>B1</td>
                  <td>B2</td>
                  <td>B3</td>
                  <td>B4</td>
              </tr>
              <tr>
                  <th>C</th>
                  /* создание слияние ячейки с нижней по уровню */
                  <td rowspan="2">C1+D1</td>
                  <td>C2</td>
                  <td>D2</td>
                  <td rowspan="2">C4+D4</td>
              </tr>
              <tr>
                  <th>D</th>
                  <td>D2</td>
                  <td>D3</td>
              </tr>
          </tbody>
          </table>
          
        
          
      /* --- css--- */

      table, th, td {
        width: 500px; /* задаем размер таблице */
        border-collapse: separate; /* определяем вид границы */
        border: 2px solid #222; /* задаем правила для границы */
      }

      caption { font-weight: 900; /* задаем жирность для заголовка таблицы */ }

      td { text-align: center; /* оцентрируем значения в ячейках */ }

      tr:hover { background-color: lightgrey; /* при наведении выделении строки */ }

      th:hover, 
      td:hover { background-color: tomato; /* при наведении выделение ячейки */ }
