Reset [Сброс]

Таблица стилей сброса (или CSS reset) - это набор правил CSS, используемых для очистки форматирования HTML-элементов браузера по умолчанию, устраняя потенциальные несоответствия между различными браузерами. Это также не позволяет разработчикам неосознанно полагаться на стиль браузера по умолчанию и заставлять их четко указывать стиль, который они хотят применить на странице.
— взято с Wikipedia
