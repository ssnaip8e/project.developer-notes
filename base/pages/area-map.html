Area/Map [Карта-изображение]

Создание навигации через сетку координатов описывающих области на 

              
  /* ====================
         Структура 
  ====================== */

  /* теги */

  map – создание карты
  area – создание области
  img – создание графического изображения

  /* атрибуты */

  shape  – задает форму области:
    [rect] - прямоугольная область x1, y1, x2, y2 
      (x1, y1 - координаты левого верхнего угла, 
      x2, y2 - координаты правого нижнего угла)
    [circle] - круглая область x, y, r 
      (x, y - координаты центра, r - радиус)
    [poly] - полигональная область x1, y1, x2, y2, ... xn, yn 
      (xi, yi - координаты i-й вершины полигона)

  coords – задает размер области
  href – задает адресс ссылки
  target – открывает ссылку в заданном значении атрибута
  title – задает подсказку для ссылки
  download – задает скачивание файла
  usemap – задает название графическому изображению для создания карты
  name – устанавливает название графического изображения созданное для карты
  nohref – делает область неактивной. Обязательно область [nohref] должна идти первой.

    <area shape="circle" coords="50,50,20" title="Белый круг" nohref>
    <area shape="circle" coords="50,50,50" title="Синий круг" href="#blue_link">

  /* ----------------------
         Образец
  ----------------------- */
  
  <img src="planets.gif" width="145" height="126" alt="Planets"
  usemap="#planetmap">
  
  <map name="planetmap">
    <area shape="rect" coords="0,0,82,126" href="sun.htm" alt="Sun">
    <area shape="circle" coords="90,58,3" href="mercur.htm" alt="Mercury">
    <area shape="circle" coords="124,58,8" href="venus.htm" alt="Venus">
  </map>
