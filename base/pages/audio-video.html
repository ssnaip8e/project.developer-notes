Audio/Video [Аудио/Видео]

Добавление медиа-контента. В форматах видео- и аудио-файлов. 

              
  /* ====================
         Структура 
  ====================== */

  /* теги */

  audio – создание аудио
  video – создание видео

  /* атрибуты */

  src      – задает ссылку на аудио/видео-файл (ресурс)
  controls – выводит элементы для управления ролика/трека (панель управления)

    <audio src= "media/the_beatles_help.mpЗ" 
    controls></audio>

  autoplay – запускает воспроизведение сразу после открытия страницы
  loop     – задает цикл воспроизведения (количество проигрываний)
  muted    – предписывает изначально приглушение звука
  poster   – задает ссылку на файл с постером - графическим изображением

    <video src="media/the_beatles_yesterday.mp4"
    poster="images/the_beatles.png"></video>

  height   – задает высоту ролика/трека в выведенного на страницу
  width    – задает ширину ролика/трека в выведенного на страницу
  preload  – управляет предварительной загрузкой ролика/трека:
    [none] - значение не загружать ролик/трек после 
        загрузки страницы, 
        однако в этом случае в панели воспроизведения 
        изначально вообще ничего не будет выведено

        <video src="media/the_beatles_yesterday.mp4" 
        preload="none"></video>

    [metadata] - значение загружать ролик/трек, 
        в котором хранятся его геометрические размеры, 
        продолжительность и первый кадр, 
        который будет выведен в панели воспроизведения 
        (поведение по умолчанию)
        
    [auto] - значение загрузить весь ролик, указывать 
        это значение имеет смысл только коротким роликам - 
        иначе загрузка затянется надолго

/* -------------------------------------------------------------------------
   Если указан атрибут [autoplay], значение атрибута [preload] игнорируется
   
   Для размещения на веб-страницах следует использовать видеоролики 
   со следующими параметрами: 
      - МР4;
      - MPEG-4 AVC (также известен как AVC и Н.264);
      - МРЗ.
--------------------------------------------------------------------------- */
