Float [Обтекание]

Свойство определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон.

Float не предназначен для создания сеток или табличной раскладки! Раньше не было другого способа разбить контент на колонки и поставить их рядом друг с другом. Но сейчас для этого есть флексы и гриды, а float используется только для «обтекания».

Изменяет элемент на блочную модель. Применяется ко всем элементам (за исключением абсолютно позиционированных).

              
/* ========================================
  Плавающий элемент
======================================== */

  float: left;  
  float: right;
  float: none; 
  float: inherit;

Clear [Очистка]

Для похожей цели появилось значение flow-root для свойства display — оно как бы изолирует обтекание. Достаточно применить его к блоку, внутри которого есть элемент с float — и влияние float не будет распространяться вне этого блока. А относительно соседей он будет вести себя как обычный блочный элемент со статичным позиционированием.

Для подобной изоляции можно указать блоку свойство hidden. Отличие состоит в том, что flow-root позволяет использовать свойство overflow: visible.

              
/* ========================================
  Очистка обтекания
=========================================== */

  clear: both;


/* ---------------------------------------
  Популярные фишки по очистке обтекания
------------------------------------------ */

[v1]
  .clearfix::after {
    display: block;
    content: " ";
    clear: both;
  }

[v2]
  .clearfix::before,
  .clearfix::after {
    display: table;
    content: " ";
  }
  .clearfix::after {
    clear: both;
  }
