Form [Форма]

Элемент страницы, предназначенный для занесения в него пользовательских данных.

              
  /* ====================
         Структура 
  ====================== */

  /* теги */

  audio – создание аудио
  video – создание видео

  /* атрибуты */

  src      – задает ссылку на аудио/видео-файл (ресурс)
  controls – выводит элементы для управления ролика/трека (панель управления)
