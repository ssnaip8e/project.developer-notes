Table [Таблица]

Таблицы HTML позволяют веб-разработчикам упорядочивать данные в строки и столбцы.

Задать границы элементам [tr], [thead], [tfoot] и [tbody] нельзя, поэтому задавайте их тегам [table], [th] или [td].

Ширина таблицы по умолчанию подстраивается под контент внутри, если не задавать дополнительные CSS-свойства. Это приводит к определённым сложностям на адаптивных сайтах. Если контент не помещается на маленький экран, то таблица не сжимается, у неё появляется горизонтальный скролл. У этой проблемы есть несколько потенциальных решений: скрывать не первостепенную информацию для пользователей мобильных устройств или перестраивать отображение таблицы, например, при помощи свойства [display].

                  
/* ===========================
   Структура таблицы
============================= */

   /* Тэги */

   table    – создание таблицы
   caption  – заголовок таблицы
   tr       – формирование строки таблицы
   th       – создание ячейки шапки
   td       – создание ячейки







/* ============================
   Образец таблицы #1
============================= */



   Таблица 1
   

      Имя
      Год
      Город
   

   

      Петр
      1992
      Минск
   

   

      Василий
      1991
      Могилев
   

 


 <table border="1">
   <caption>Таблица 1</caption>
   <tr>
      <th>Имя</th>
      <th>Год</th>
      <th>Город</th>
   </tr>
   <tr>
      <td>Петр</td>
      <td>1992</td>
      <td>Минск</td>
   </tr>
   <tr>
      <td>Василий</td>
      <td>1991</td>
      <td>Могилев</td>
   </tr>
</table>

                  
               
                  
/* ===========================
   Структура таблицы
============================= */

   /* Логические секции */

   thead    – секция шапки таблицы
   tbody    – секция тело таблицы
   tfoot    – секция подвала таблицы

   /* Атрибуты */

   colspan  – слияние ячеек по горизонтали
   rowspan  – слияние ячеек по вертикали
   colgroup – назначает группу столбцов для форматирования
   col      – задает свойства столбца в colgroup


/* ============================
   Образец таблицы #2
============================= */



   
     
     
   
   
      

         1
         2
         3
      

   
   
      

         A
         B
         C
      

      

         Четверг
         Пятница
         Суббота
      

   
 


 <table>
   <colgroup>
      <col span="2" style="background-color:green">
      <col style="background-color:red">
   </colgroup>
   <thead>
      <tr>
         <td>1</td>
         <td>2</td>
         <td>3</td>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>A</td>
         <td>B</td>
         <td>C</td>
      </tr>
      <tr>
         <td>Четверг</td>
         <td>Пятница</td>
         <td>Суббота</td>
      </tr>
   </tbody>
 </table>

                     
                  
                  
/* ========================
   Дизайнерский приём 
=========================*/
/* -------------------------------------------------------------------------------
   Подсветка строк таблицы через одну. 
   Это помогает считывать длинные таблицы, глазу есть за что зацепиться.
   Например, сделаем каждую вторую строку с коричневым фоном. 
   Для этого понадобится всего одно CSS-правило с псевдоклассом [:nth-child()]
-------------------------------------------------------------------------------- */

   tbody tr:nth-child(odd) {
      background-color: #663613;
   }

/* --------------------------------------------------------------------
   Можно сделать так, чтобы строка с заголовками колонок прилипала 
   при прокрутке длинной таблицы. 
   Это удобно, если данных много и пользователь может просто забыть, 
   в какой колонке какие данные.
-------------------------------------------------------------------- */

   th {
      position: -webkit-sticky;
      position: sticky;
      top: 0;
      z-index: 1;
   }                     

/* --------------------------------------------------------------------
   Не забудьте добавить position: relative для родителя. 
   Заодно подстрахуемся и сделаем прилипающими только заголовки 
   в шапке таблицы.
-------------------------------------------------------------------- */
   table {
      position: relative;
   }
   
   thead th {
      position: sticky;
      position: -webkit-sticky;
      top: 0;
      z-index: 1;
   }

/* --------------------------------------------------------------------
   Задайте фон заголовкам, чтобы текст ячеек не был виден при прокрутке. 
   А чтобы избавиться от линий между ячейками, зададим для всей таблицы 
   свойство [border-collapse: collapse]
-------------------------------------------------------------------- */

   table {
      position: relative;
      border-collapse: collapse;
   }
   
   thead th {
      position: -webkit-sticky;
      position: sticky;
      top: 0;
      z-index: 1;
      background-color: #FF8630;
   }
